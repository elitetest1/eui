Elite UI v8.4

  • OneUI 8.5 apps
  • S25 Ultra wallpapers
  • More optimization
  • Nuked logging/tracing using a new way
  • Added MultiControl
  • Added MyDevice
  • Added MCFDeviceSync
  • Added HWResourceShare
  • Disabled ATrace: Turned off various Android system and app-level tracing features.
  • Disabled Graphics Tracing: Disabled tracing for EGL, Vulkan, and Skia (the 2D graphics engine).
  • Disabled Hardware UI (HWUI) Tracing: Turned off tracing for the hardware-accelerated UI renderer, including GPU resource tracking.
  • Disabled General & Third-Party Logging: Disabled some master log switches and logging from third-party apps.
  • Disabled HDR Logging: Turned off specific logging for HDR10+ video processing.
  • Disabled ML Tracing: Disabled tracing for TensorFlow Lite (on-device machine learning).
  • Disabled Document Scanner Logging: Turned off logging for a specific document scanning service.
  • Disabled Incremental FS Logging: Disabled logging related to the Incremental File System (used for app loading).
