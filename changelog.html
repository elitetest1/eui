Changelog

Full release history for Elite UI — every build, every fix.

Latest Release

Elite UI v8.5

  • Initial release of OneUI 8.5 based on Android 16 QPR2.
  • VIP Version: Includes constant updates and bug fixes.
  • Free Version: A courtesy release that does not receive patches or bug fixes.

Elite UI v8.4

  • OneUI 8.5 apps
  • S25 Ultra wallpapers
  • More optimization
  • Nuked logging/tracing using a new way
  • Added MultiControl, MyDevice, MCFDeviceSync, HWResourceShare
  • Disabled ATrace: Turned off various Android system and app-level tracing features.
  • Disabled Graphics Tracing: EGL, Vulkan, and Skia (2D graphics engine).
  • Disabled Hardware UI (HWUI) Tracing: GPU resource tracking.
  • Disabled General & Third-Party Logging.
  • Disabled HDR Logging: Specific logging for HDR10+ video processing.
  • Disabled ML Tracing: TensorFlow Lite (on-device machine learning).
  • Disabled Document Scanner Logging.
  • Disabled Incremental FS Logging.

Elite UI v8.3.1

  • Nuked Samsung's Low Memory Killer Daemon.
  • Switched to AOSP's Low Memory Killer Daemon.
  • Fixed HDR rebooting issues in r9s/o1s/t2s.
  • Adapted LMK values for better RAM management.
  • Disabled App Freezer. Enabled erofs_hipri.
  • Fixed edge lightning corners.
  • Fixed audio modes / quality.
  • Finally fixed camera 30x processing.
  • Fixed light mode glitch.
  • Added nearby devices, wifi guider, contextual search overlay from s25fe.
  • Added game tools, game optimizing service, battery bypass in game tools.
  • Re-added smb_server. Default BBR.
  • Disabled powerdelivery charger hv. Enabled EROFS_FS_PCPU_KTHREAD. CRC enabled.
  • Backport percpu threads. Enabled battery bypass.
  • Increased USAP Pool for faster app startup (Zygote).
  • Enabled "Fast Dormancy" to reduce modem battery consumption.
  • Reduced Wi-Fi scanning frequency. Enabled modem deep sleep mode.
  • Improved device power-on time (Quick Power-On).
  • UI smoothness, rendering tweaks, reduced screen composition latency.

  • Fixed camera post-processing in 10x/20x/30x zoom.
  • Added AppLock, OutDoor mode, Adaptive color tone.
  • Added ROM Logo in software info.
  • Fixed Fingerprint issue in S21/S21+ (o1s/t2s).
  • Proton+ 5.F. Added Battery Info & Health in settings.

  • Various new features, fixes and tweaks.
